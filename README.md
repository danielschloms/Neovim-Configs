# My personal Neovim/NvChad Config

## Install

+ Install neovim (>= 0.9) and NvChad (requires `ripgrep`)
+ `cd ~/.config/nvim/lua/`
+ `rm -rf custom/
+ `git clone git@github.com:danielschloms/NvChad-Config.git custom`

## Supported languages

+ C++
+ Rust
